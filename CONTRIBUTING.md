# Contributing Rules

## Core rule
Do not build ahead of the active stage.

## Every feature must include
- Outcome
- Scope
- Non-goals
- Acceptance criteria

## Every stage gate must include
- Completed checklist
- Attached evidence
- Remaining risks

## Claude Code usage rule
Use the prompt file for the active stage first. Then break work into feature issues and have Claude Code implement one scoped feature at a time.
