# Feature Prompt — Auth and Organization Onboarding

Implement authentication and organization onboarding for the HVAC SaaS.

## Scope
- signup/login flow
- organization creation flow
- onboarding status persistence
- next-step checklist after onboarding

## Must include
- secure auth pattern
- organization ownership assignment
- basic onboarding UI and server actions/API routes
- analytics events: user_signed_up, organization_onboarding_started, organization_onboarding_completed

## Do not include
- billing subscriptions
- team invites beyond minimal placeholders
