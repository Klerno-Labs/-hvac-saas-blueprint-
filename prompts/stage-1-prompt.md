# Stage 1 Prompt

Act as a startup validation operator.
Goal: create a no-call validation sprint for an HVAC quote-to-payment product.
Deliverables:
1. Four-email outreach sequence.
2. Two-minute Loom script.
3. Founding customer / concierge offer.
4. Tracking framework for reply rate, trial starts, first invoices, first payments, objections, and support time.
5. Pass, warning, fail thresholds.
Constraints:
- No live sales calls by default.
- Must optimize for async outreach and self-serve trial.
- Language must be direct, low-hype, and outcome-focused.
