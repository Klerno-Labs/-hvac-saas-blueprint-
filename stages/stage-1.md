# Stage 1 — Validation Sprint

## Objective
Get the strongest possible proof before full build through async outreach, landing page, Loom, and real workflow interest.

## Deliverables
- landing page
- pricing block
- 2-minute Loom
- outreach sequence
- tracking sheet/dashboard
- objections log

## Checklist
- [ ] landing page published
- [ ] Loom recorded
- [ ] 4-email sequence ready
- [ ] first target list built
- [ ] first 50 emails sent
- [ ] first 100 emails sent
- [ ] results logged
- [ ] objections categorized
- [ ] pass/warning/fail decision made

## Evidence required to close
- outreach numbers attached
- screenshots or links attached
- conclusion written: continue, iterate, or pivot
